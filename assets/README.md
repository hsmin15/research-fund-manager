# Assets 폴더

이 폴더에 메뉴얼 이미지 파일을 저장하세요.

## 필요한 이미지 파일

다음 이미지 파일들을 이 폴더에 추가하면 됩니다:

1. `meeting-pre-manual.png` - 회의비 사전신청 메뉴얼
2. `meeting-manual.png` - 회의비 신청 메뉴얼
3. `activity-manual.png` - 연구활동비 신청 메뉴얼
4. `materials-manual.png` - 연구재료비 신청 메뉴얼

## 사용 방법

이미지를 추가한 후, `professor.html` 파일에서 다음과 같이 이미지를 표시할 수 있습니다:

```html
<img src="assets/meeting-pre-manual.png" alt="회의비 사전신청 메뉴얼" style="max-width: 100%; height: auto;">
```
